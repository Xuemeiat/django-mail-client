# Mail Project (Django + JavaScript)

This is a fully functional email client built as part of the CS50 Web Programming course.
The project includes:

- Inbox view
- Sent mailbox
- Archived mailbox
- Compose email
- Reply feature
- Mark emails as read/unread
- Dynamic JavaScript interface
- Django backend API

## Features

### ✔ Inbox View
Displays all received emails, marks read/unread, and allows opening full messages.

### ✔ Compose & Send
Send emails using a clean and responsive form.

### ✔ Archive / Unarchive
Move emails to archive and restore them.

### ✔ Reply
Automatically fills subject and body with quoted text.

## Technologies Used
- Python / Django
- JavaScript (DOM manipulation + fetch API)
- HTML / CSS
- Bootstrap
